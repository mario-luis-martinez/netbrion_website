### Cómo publicar el sitio (rápido y sin complicaciones) 🚀

1. Descargar el repositorio completo desde GitHub
   (botón **Code → Download ZIP**) o clonarlo.

2. Extraer los archivos en tu computadora.

3. Subir **todo el contenido** al directorio público del hosting:

   * normalmente `public_html`
   * o `www`
   * o `htdocs`

4. Asegurarse de que los archivos queden así en la raíz:

   * `index.html`
   * `sitemap.xml`
   * `robots.txt`
   * `.htaccess`
   * carpetas `css`, `js`, `images`, etc.

⚠️ Importante:
No subir una carpeta contenedora.
Los archivos deben quedar **directamente en la raíz del dominio**.

5. Abrir el dominio en el navegador.
   El sitio ya debería estar funcionando.

### Recomendación después de publicar

* Enviar `sitemap.xml` a Google Search Console
* Verificar que el dominio cargue con HTTPS
* Probar versión móvil

Listo. El sitio queda online y preparado para indexación.
